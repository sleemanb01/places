~~ This is a MERN stack website

~ The site allow users to share their favorite places

* It is developed with: ReactJs, Typescript, NodeJs, ExpressJs, MongoDB

* The site developed with consideration of high performance, and efficient states.

~ Things to be proud of : 

* A reusable form hook that can handle any number of inputs, with the help of a reducer

* A http hook which handles all HTTP requests and return handling functions for specific uses,
it also contain a cleanup function for aborted requests

* A robust security, hashing and tokens
